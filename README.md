# Brain Games
Минималистичные консольные игры для тренировки мозга.

## Установка и запуск
1. Склонируйте репозиторий: `git clone https://github.com`
2. Установите зависимости: `npm install`
3. Создайте символическую ссылку: `npm link`

## Игры
### Brain-GCD (НОД)
Нахождение наибольшего общего делителя.
Команда для запуска: `brain-gcd`
![Демонстрация игры](./assets/gcd-demo.png)

### Brain-Progression (Прогрессия)
Поиск пропущенного числа.
Команда: `brain-progression`
![Демонстрация игры](./assets/progression-demo.png)
